VisBugEditor - Ignore List Feature

Usage Example:

const editor = new VisBugEditor({
  container: document.getElementById('editable-area')
});

// Set elements to ignore during hover and selection
editor.ignoreList([".header", ".ignored-element"]);

// Get current ignore list
const ignored = editor.getIgnoreList();
